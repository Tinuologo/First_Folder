The Basics

As per the norm, the tutorials began with the basic and eventually ended with the advanced. Here are some of the lessons learned from the tutorials in the basic section. Most elements require a <start tag> and an <end tag>. Attributes provide additional information about HTML elements and all elemtents can have attributes. Some examples of attributes are href for links, img for images, and style for specify the styling of an element, like color, font, size etc.
